## 📂 内容分类与关联说明（Repositories 分类说明）

> 四个板块并不是独立存在，而是构成一个从实验理解到工程落地的完整路径：**实验框图 → 实验技巧 → 软件优化 → 工程与商业应用**。

### 🧩 1. Experimental Architecture（实验框图）

**自动化控制实验框图与信号链存档**
聚焦论文与教学实验中的硬件搭建、信号连接与信号流程，系统整理实验框图，用于经验沉淀、快速理解与工程复现，为后续技巧与优化提供基础。

---

### 🧩 2. Control Practice（实验技巧）

**控制实验实操技巧与避坑经验**
围绕自动化控制实验中的硬件连接、模块配置与实际操作，分享常见问题、避坑思路与工程经验，帮助实验更稳、更快落地，同时与实验框图形成直接关联，使经验可落地。

---

### 🧩 3. Software Optimization（软件优化）

**Simulink 控制系统优化与高级用法**
聚焦 Simulink 在自动化控制中的性能优化与高级特性，包括多线程、CPU 调度与模型结构优化，提升系统运行效率与工程可用性，将实验技巧与框图逻辑抽象为可复用的软件方法。

---

### 🧩 4. Engineering Projects（工程与商业应用）

**自动化控制课题与工程化应用**
面向产学合作场景，将基于 Simulink 的控制算法快速转化为可运行的硬件系统，支持课题验证、原型实现与工程落地。它是前面三者的综合体现，展示从实验到工程化应用的完整路径。
